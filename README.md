# React Listings

Minimal starter for a listings web app built with React, Vite, TypeScript, Tailwind, React Router, and Supabase.

## Setup

1. Install Node.js 20+.
2. Install dependencies:

   ```bash
   npm install
   ```

3. Copy environment variables:

   ```bash
   copy .env.example .env
   ```

4. Add your Supabase values to `.env`.

## Run

```bash
npm run dev
```
